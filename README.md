## Requirements

> [!WARNING]
> Requires Neovim 0.10.0+.

## Install

Via [lazy.nvim](https://github.com/folke/lazy.nvim):

```lua
{ 'mistweaverco/retro-theme.nvim' },
```
See [configuration options](#configuration) for more information.

## Configuration

```lua
{
  'mistweaverco/retro-theme.nvim',
  opts = {
    italic_comments = true,
    disable_cache = false,
    hot_reload = false,
  }
},
```
